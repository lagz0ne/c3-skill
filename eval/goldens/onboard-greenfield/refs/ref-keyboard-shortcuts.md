# Keyboard Shortcuts

## Goal

Define standard keyboard shortcuts for canvas navigation and manipulation, ensuring consistent user experience across all spatial interactions.

## Canvas Navigation

| Shortcut | Action | Context |
|----------|--------|---------|
| Space + Drag | Pan canvas | Any |
| Scroll wheel | Zoom in/out | Any |
| Cmd/Ctrl + 0 | Reset zoom to 100% | Any |
| Cmd/Ctrl + 1 | Fit all to view | Any |
| Cmd/Ctrl + 2 | Fit selection to view | Selection |
| Arrow keys | Pan incrementally | Any |
| Shift + Scroll | Horizontal pan | Any |

## Selection

| Shortcut | Action | Context |
|----------|--------|---------|
| Click | Select single | Concept |
| Shift + Click | Add to selection | Concept |
| Cmd/Ctrl + Click | Toggle selection | Concept |
| Cmd/Ctrl + A | Select all | Any |
| Escape | Deselect all | Selection |
| Tab | Select next | Selection |
| Shift + Tab | Select previous | Selection |

## Concept Operations

| Shortcut | Action | Context |
|----------|--------|---------|
| N | New concept | Canvas |
| Delete/Backspace | Delete selected | Selection |
| Cmd/Ctrl + D | Duplicate | Selection |
| Cmd/Ctrl + C | Copy | Selection |
| Cmd/Ctrl + V | Paste | Any |
| Cmd/Ctrl + X | Cut | Selection |
| Enter | Edit concept | Single selection |

## Movement

| Shortcut | Action | Context |
|----------|--------|---------|
| Arrow keys | Nudge position | Selection |
| Shift + Arrow | Large nudge | Selection |
| Cmd/Ctrl + [ | Send backward | Selection |
| Cmd/Ctrl + ] | Bring forward | Selection |

## View Controls

| Shortcut | Action | Context |
|----------|--------|---------|
| Cmd/Ctrl + + | Zoom in | Any |
| Cmd/Ctrl + - | Zoom out | Any |
| G | Toggle grid | Any |
| M | Toggle minimap | Any |

## AI and Chat

| Shortcut | Action | Context |
|----------|--------|---------|
| / | Open AI chat | Any |
| Cmd/Ctrl + K | Quick AI action | Any |
| Escape | Close chat panel | Chat open |

## Accessibility

| Shortcut | Action | Context |
|----------|--------|---------|
| ? | Show shortcuts help | Any |
| Cmd/Ctrl + , | Open settings | Any |

## Platform Variations

| Mac | Windows/Linux |
|-----|---------------|
| Cmd | Ctrl |
| Option | Alt |
| Control + Click | Right-click |
