# Review Lenses

Use these lenses when comparing outputs across Codex, Claude, and Kilo.

| Lens | What To Look For |
| --- | --- |
| Skill adherence | Did the agent use local C3 commands and follow the supplied skill flow, or did it improvise generic docs? |
| Growth mechanics | Did it preserve the rung model and migrate docs when complexity increased? |
| Boundary clarity | Are frontend, backend, integration, and database boundaries separate and meaningful? |
| Cross-container reasoning | Are flows like reservation, receiving, cycle count, and integration traced across containers? |
| Gardening discipline | Did the agent clean and verify docs after growth, or leave partial/ragged artifacts? |
| Model spread | Do free/low-cost Kilo models fail the same dimensions as paid models, or are failures agent-specific? |
